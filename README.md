(c)鼠标和键盘的交互

1.相机：

移动： 前后左右-WASD ，空格-提升高度 ，Shift-降低高度 ，R-回到最初视角 

鼠标控制第一人称视角 ，Z键-捕捉/不捕捉鼠标

2.光源：

移动：前后左右- IKJL ，上下-UO

3.网格通过 1 2 切换

移动：前后左右-↑↓←→  ，上下-小键盘12 ，前后旋转-小键盘85 ，左右选择-小键盘46

小键盘按0-全体逆时针旋转
